Here are some coding projects that have kept me busy on the side:

Espresso Patronum
Simple web application to find nearby coffee shops. Uses the Google Maps Places API to find nearby results and sort them by rating. Users can scroll through the results, get directions in Google Maps, and view ratings from Yelp. Emphasis on clean design and avoiding feature-creep.

Find some coffee

#tweeter
Sometimes you need to craft the perfect 140 character tweet. This tool takes Twitter back to basics. Born from an idea to build a tweet composer that points out where you can reduce your character count. Still in progress.

Compose a Tweet

WhereTo?
My first indepedent coding project, recreating an older school project. Uses jQuery and CSS to present a prototype mobile application for better navigating public transit. The school project focused on design thinking and user needs, this reimplementation focused on aesthetics and animations. Focus on information display and spatial design.
